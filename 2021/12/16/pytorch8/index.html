1、SoftMax — dim应当如何填？

官方文档：https://pytorch.org/docs/stable/generated/torch.nn.functional.softmax.html?highlight=softmax#torch.nn.functional.softmax

1）标准用法：

xxxxxxxxxx
torch.nn.functional.softmax(input, dim)
（Applies a softmax function.）

2）依赖库

xxxxxxxxxx
import torch
import torch.nn.functional as F

3）简单示例

xxxxxxxxxx
a=torch.rand(3,16,20) # 我们看一个三维的tensor
b=F.softmax(a,dim=0)  
# dim = 0代表 输出的是在dim=0维上的概率分布，也就是说 对于任意的i,j (i是dim=1上的某个坐标，j是dim=2上的某个坐标)，通过softmax之后，要求 a[0][i][j] + a[1][i][j] + a[2][i][j] = 1 【此处仅有3项相加，原因是a向量dim=0维度 长度为3，仅有3个元素。】

4）一个更为一般的示例

xxxxxxxxxx
a=torch.rand(x1,x2,x3,x4,……,xn) # 我们看一个n维的tensor
b=F.softmax(a,dim=m)


# dim = m代表 输出的是在dim=m维上的概率分布，也就是说 通过softmax之后，对于任意的dim!=m的维度的一个定位：i1,i2,i3,……im-1,im+1,……in，需要满足如下公式 
#  a[i1][i2]……[im-1][0][im+1]……[in] + 
#  a[i1][i2]……[im-1][1][im+1]……[in] +
#  a[i1][i2]……[im-1][2][im+1]……[in] +
#  …… +
#  a[i1][i2]……[im-1][xm][im+1]……[in] = 1 即可

3、Permute函数

1）标准用法：

xxxxxxxxxx
torch.permute(input, dims) → Tensor
- parameters:
  input (Tensor) – the input tensor.
  dims (tuple of python:ints) – The desired ordering of dimensions

2）依赖库：

xxxxxxxxxx
import torch

3）简单示例：

xxxxxxxxxx
x = torch.randn(2, 3, 5)
print(x.size())
# torch.Size([2, 3, 5])
print(torch.permute(x, (2, 0, 1)).size())
# torch.Size([5, 2, 3])
# 我们可以看到：  [dim0,dim1,dim2,dim3,dim4]  经过permute(2,0,1,4,3)以后：
# （ permute的含义： 现在的第0维应当为原来的第2维，现在的第1维应当为原来的第0维，…… ）
# 会变成如下形式。[dim2,dim0,dim1,dim4,dim3]

6、nn.functional.pad

官方文档：https://pytorch.org/docs/stable/generated/torch.nn.functional.pad.html?highlight=pad#torch.nn.functional.pad

1）标准用法：

xxxxxxxxxx
torch.nn.functional.pad(input, pad, mode='constant', value=0.0)
-   input (Tensor) – N-dimensional tensor
- pad (tuple) – m-elements tuple, where m/2 ≤ input dimensions and m is even.
- mode – 'constant', 'reflect', 'replicate' or 'circular'. Default: 'constant'
- value – fill value for 'constant' padding. Default: 0


对于padding-size的说明：
    填充尺寸用来填充输入的某些维度，从最后一个维度开始进行描述。比如说:
    如果想要填充最后一个维度，那就设置为(padding_left,padding_right)
    如果想要填充最后两个维度，那就设置为(padding_left,padding_right,padding_top,padding_bottom)

2）依赖库：

xxxxxxxxxx
import torch.nn.function as F

3）简单示例：

xxxxxxxxxx
import torch
import torch.nn.functional as F
t2d = torch.rand(2, 2)
p1d = (1, 1)
print(t2d)
out = F.pad(t2d, p1d, "constant", 0)
print(out.size())
print(out)
"""
tensor([[0.6359, 0.0629],
        [0.5814, 0.5980]])
torch.Size([2, 4])
tensor([[0.0000, 0.6359, 0.0629, 0.0000],
        [0.0000, 0.5814, 0.5980, 0.0000]])
"""

4）扩展示例：

xxxxxxxxxx
>>> t4d = torch.empty(3, 3, 4, 2)
>>> p3d = (0, 1, 2, 1, 3, 3) # pad by (0, 1), (2, 1), and (3, 3)
>>> out = F.pad(t4d, p3d, "constant", 0)
>>> print(out.size())
torch.Size([3, 9, 7, 3])

 

7、flatten

官方文档：https://pytorch.org/docs/stable/generated/torch.flatten.html#torch.flatten

1）标准用法：

xxxxxxxxxx
torch.flatten(input, start_dim=0, end_dim=- 1) → Tensor
  - input (Tensor) – the input tensor.
  - start_dim (int) – the first dim to flatten
  - end_dim (int) – the last dim to flatten

2）依赖库：

xxxxxxxxxx
import torch

3）简单示例：

xxxxxxxxxx
import torch


t2d = torch.rand(2, 2, 2)
print(t2d, t2d.size())
out = torch.flatten(t2d, 1)  # 从第1维开始，展平后续所有维度
print(out, out.size())
"""
tensor([[[0.4151, 0.8645],
         [0.1289, 0.2260]],
        [[0.3302, 0.0393],
         [0.9822, 0.5305]]]) torch.Size([2, 2, 2])
tensor([[0.4151, 0.8645, 0.1289, 0.2260],
        [0.3302, 0.0393, 0.9822, 0.5305]]) torch.Size([2, 4])
"""

4）扩展示例:

xxxxxxxxxx
Q1、现在有一个向量t，其size为 [A,B,C,D,E,F,G] ,然后我们对其应用flatten: torch.flatten(t,2,4),
那么size会变成？
Answer: [A,B,C*D*E,F,G]


Q2、现在有一个向量t，其size为 [A,B,C,D,E,F,G] ,然后我们对其应用flatten: torch.flatten(t,-2),
那么size会变成？
Answer: [A,B,C,D,E,F*G] 等效于应用torch.flatten(t,5) 等效于应用torch.flatten(t,5,6)


Q3、现在有一个向量t，其size为 [A,B,C,D,E,F,G] ,然后我们对其应用flatten: torch.flatten(t),
那么size会变成？
Answer: [A*B*C*D*E*F*G] 


Q3、现在有一个向量t，其size为 [A,B,C,D,E,F,G] ,然后我们对其应用flatten: torch.flatten(t,1),
那么size会变成？
Answer: [A,B*C*D*E*F*G]

 

8、Transpose

官方文档：https://pytorch.org/docs/stable/generated/torch.transpose.html#torch.transpose

1）标准用法：

xxxxxxxxxx
torch.transpose(input, dim0, dim1) → Tensor


-   Returns a tensor that is a transposed version of input. The given dimensions dim0 and dim1 are swapped.
- input (Tensor) – the input tensor.
- dim0 (int) – the first dimension to be transposed
- dim1 (int) – the second dimension to be transposed


注：此处的转置通用于高阶矩阵，函数将会对dim0和dim1这两个维度形成的平面进行转置，交换dim0和dim1,而不影响其他维度。

2）依赖库：

xxxxxxxxxx
import torch

3）简单示例：

xxxxxxxxxx
t2d = torch.rand(2, 3)
print(t2d, t2d.size())
out = torch.transpose(t2d, 0, 1)
print(out, out.size())


"""
tensor([[0.5228, 0.2990, 0.1111],
        [0.4857, 0.4479, 0.6637]]) torch.Size([2, 3])
        
tensor([[0.5228, 0.4857],
        [0.2990, 0.4479],
        [0.1111, 0.6637]]) torch.Size([3, 2])
"""

4）扩展示例：

xxxxxxxxxx
在高维的矩阵中，不要太过注重打印数据显示的变化形式，从维度长度的变化入手，即可清晰的搞懂


Q1、现在有一个向量t，其size为 [A,B,C,D,E,F,G] ,然后我们对其应用transpose: torch.transpose(t,2,4),
那么size会变成？


Answer: [A,B,E,D,C,F,G]  可以看到，从size上来讲其实就是交换了第2维和第4维的长度。从数据上来讲，你可以想象其找到了第2维和第4维形成的数据平面，然后应用了正常的转置。

 

9、Cat函数

官方文档：https://pytorch.org/docs/stable/generated/torch.cat.html?highlight=cat#torch.cat

1）标准用法：

xxxxxxxxxx
torch.cat(tensors, dim=0, *, out=None) → Tensor


- Concatenates the given sequence of seq tensors in the given dimension. All tensors must either have the same shape (except in the concatenating dimension) or be empty.


- tensors (sequence of Tensors) – any python sequence of tensors of the same type. Non-empty tensors provided must have the same shape, except in the cat dimension.
- dim (int, optional) – the dimension over which the tensors are concatenated


注：tensors参数需要输入一个tensor的序列，所有的需要拼接的tensors需要有同样的形状维度（除了拼接的那一维）
dim指定要拼接的维度

2）依赖库：

xxxxxxxxxx
import torch

3）简单示例：

xxxxxxxxxx
import torch
x = torch.randn(2, 3)
print(x, x.size())
out = torch.cat((x, x, x), 0)
print(out, out.size())
"""
tensor([[-1.2926, -0.3155,  0.5966],
        [ 0.8921, -2.0986,  0.0748]]) torch.Size([2, 3])
        
tensor([[-1.2926, -0.3155,  0.5966],
        [ 0.8921, -2.0986,  0.0748],
        [-1.2926, -0.3155,  0.5966],
        [ 0.8921, -2.0986,  0.0748],
        [-1.2926, -0.3155,  0.5966],
        [ 0.8921, -2.0986,  0.0748]]) torch.Size([6, 3])
"""

4）扩展示例：

xxxxxxxxxx
Q1、现在有一个向量t1，其size为 [A,B,C,D] ,向量t2，其size为[A,B,E,D] 然后我们对其应用cat: torch.cat((t1,t2),2),那么size会变成？


Answer: [A,B,C+E,D]


Q2、现在有一个向量t1，其size为 [A,B,C,D] ,向量t2，其size为[A,B,E,D] 然后我们对其应用cat: torch.cat((t1,t2),1),那么size会变成？


Answer: 函数会报错，因为除了维度1以外，t1和t2的维度2的shape不一样

 

10、Roll

官方文档：https://pytorch.org/docs/stable/generated/torch.roll.html?highlight=roll#torch.roll

1）标准用法：

xxxxxxxxxx
torch.roll(input, shifts, dims=None) → Tensor


    沿着给定的维数滚动张量。移动到最后一个位置以外的元素将在第一个位置重新引入。如果没有指定尺寸，张量在滚动之前将被压平，然后恢复到原来的形状。
    
- input (Tensor) – the input tensor.
- shifts (int or tuple of python:ints) – 张量的元素被移动的位置数。如果shift是一个元组，dim必须是一个相同大小的元组，并且每个维度将被相应的值滚动
- dims (int or tuple of python:ints) – Axis along which to roll

2）依赖库：

xxxxxxxxxx
import torch

3）简单示例：

xxxxxxxxxx
import torch
x = torch.tensor([1, 2, 3, 4, 5, 6, 7, 8]).view(4, 2)
print(x)
y1 = torch.roll(x, 1, 0)  # 沿着第0维（列），滚动1个元素
print(y1)
y2 = torch.roll(x, -1, 0) # 沿着第0维（列），滚动-1个元素
print(y2)
y3 = torch.roll(x, shifts=(2, 1), dims=(0, 1)) # 沿着第0维（列），滚动2个元素，沿着第1维（行），滚动1个元素
print(y3)
"""
x:tensor([[1, 2],
        [3, 4],
        [5, 6],
        [7, 8]])
y1:tensor([[7, 8],
        [1, 2],
        [3, 4],
        [5, 6]])
y2:tensor([[3, 4],
        [5, 6],
        [7, 8],
        [1, 2]])
y3:tensor([[6, 5],
        [8, 7],
        [2, 1],
        [4, 3]])
"""
